# Secure AWS Architecture

## Overview
Designed a secure cloud architecture using VPC, EC2, and S3.

## Architecture Design
- Public subnet for web server
- Private subnet for application server
- S3 bucket for secure storage

## Security Features
- IAM role with least privilege
- Security Groups restricting access
- No direct access to private resources
- Encryption enabled for data

## Key Concepts
- Defense in depth
- Least privilege
- Reduced attack surface
